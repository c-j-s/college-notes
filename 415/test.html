It can be seen that every equation in a homogenous `LS` can be written as: `sum a_i * x_i = 0 ` where `a_i\ epsilon bbR`.

  • `(0, 0, ...., 0)` is a solution for each of these. So the 0 vector must be a solution for the `bbL bbS`.

  • Let: `S = (s_1, s_2, s_3, ..., s_i)` be a solution to the ` bbL bbS `

    So, S is a solution to each of the equations, so `sum s_i * x_i = 0 `

    Multiplying by `k epsilon bbR`: `sum k * s_i * x_i = 0 `

    So `S = (k* s_1,k* s_2,k* s_3,k* ..., k* s_i)` is also a solution

    and the solution set of the LS is closed under scalar multiplaction

  • Let: `R = (r_1, r_2, r_3, ..., r_i)` be another solution to the ` bbL bbS `

    So, `sum s_i * x_i = 0 ` and `\sum r_i * x_i = 0 ` are solutions to the LS.

    Adding these gives: `sum s_i * x_i + sum r_i * x_i = 0 `

    `sum (s_i * x_i + r_i * x_i) = 0 `

    `sum (s_i + r_i ) * x_i = 0 `
